Arkanoid - Классическая игра «Арканоид», реализованная на JavaScript с использованием HTML5 Canvas.

Цель игры - Уничтожить все кирпичики, управляя платформой и отражая мячик.  
Если мяч падает вниз — проигрыш. После уничтожения всех кирпичиков — переход на следующий уровень.

Игра реализована - HTML, HTML5 (canvas), CSS, JavaScript (ES6+).

Функциональность: 
Движение платформы мышью.
Столкновение мячика с кирпичиками, платформой и стенами.
Несколько уровней с увеличением сложности.
Анимация частиц при разрушении кирпичиков.
Звуковые эффекты при столкновениях.
Счёт и уровни отображаются в интерфейсе.
Игра полностью изолирована в функции (self-invoking function).
Запуск: Открыть index.html в браузере. Через Visual Studo Code - Go Live.
