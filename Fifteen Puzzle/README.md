Fifteen Puzzle - Классическая игра «Пятнашки».
Игра представляет собой поле 3 x 3 с перемешанными плитками от 1 до 9 и одной пустой ячейкой. 
Задача игрока — расположить плитки в правильном порядке, используя минимальное количество ходов.
Игра реализована на: HTML, CSS, JavaScript.
Особенности реализации: Логика движения плиток реализована через проверку соседних ячеек. Состояние поля хранится в виде массива. Используется динамическая отрисовка DOM.
Функционал: Перемешивание игрового поля. Перемещение плиток по клику. Проверка правильной сборки.
Запуск: Открыть index.html в браузере. Через Visual Studo Code - Go Live.
